LaTeX Cheat Sheet

Экзамен ЦК

1.Куча - это…

Ответ: область памяти.

 

2.К особенности представления констант относится то, что для их размещения по умолчанию отводится…

Ответ: фактически необходимый объём памяти.



3.Строки, массивы и структуры в С++…
Ответ: могут быть переданы в качестве параметров функций, а функции могут возвращать их в основную программу.


4.Подключение заголовочного файла iomanip.h в С++ осуществляется тогда, когда используется…
Ответ: манипулятор ввода-вывода без параметров.


5.Изменения в базовом классе…
Ответ: приводят к изменениям в производных к нему классах.


6. Создание псевдонимов типов в С++ осуществляется с помощью служебного слова
Ответ: typedef.


7.В случае, если в С++ функция возвращает указатель, то принимающая его переменная…
Ответ: всегда должна иметь тип указателя.


8.К базовым алгоритмическим структурам не относятся…
Ответ: функция, переменная, машинный код.


9.Функция-элемент класса, автоматически выполняющаяся в момент создания объекта, называется…
Ответ: конструктор.


10.Переход к следующей итерации цикла позволяет выполнить оператор…
Ответ: continue (называется оператором инкремента).


11.Перегруженными в С++ называются функции, которые…
Ответ:  в одной программе имеют одинаковые имена.


12.К пользовательским типам данных в С++ относятся
Ответ: структуры(struct), классы(class).


13.Механизм, объединяющий данные и код, реализующий обработку таких данных, называется…
Ответ: Объектно-ориентированным программированием(ООП)


14.Характер языка записи алгоритма зависит от…
Ответ: Языка программирования


15.Набор символов, из которых формируются все конструкции языка программирования, называется…
Ответ: лексемы или лексический алфавит.


16.Директива #include позволяет…
Ответ: включить в текст программы содержимое указанного файла.


17. Директива #typedef позволяет…
Ответ: задать синоним для пользовательского или встроенного типа данных.


18. Служебное слово void, расположенное внутри круглых скобок на месте перечня параметров, сообщает компилятору, что функция…
Ответ: выполняет действия, но не имеет возвращаемых значений.


19.Псевдоним в С++ может быть создан…
Ответ: для переменной любого типа.


20. Выражение int_V = static_cast <int> (float_V) в С++…
Ответ: позволяет преобразовать вещественное число в целое.


21.Функция fseek() используется в С++ для…
Ответ: организации произвольного доступа к элементам двоичных файлов.


22.При i = 1234, j = 127, a = 86.531318 команда printf (“i = %06d, \n j = %d, a = %6.4f”, i, j, a) позволит вывести на экран следующую информацию…
Ответ: i = 001234, j = 127, a = 86.5313.


23.Если после служебного слова return стоит точка с запятой, то это означает, что функция…
Ответ: не имеет возвращаемого значения.


24.Если для работы функции в С++ требуется более одного параметра, то в скобках задается список этих параметров, разделенных…
Ответ: запятыми.


25.Ширину поля вывода данных в С++ при консольном выводе…
Ответ: можно задать с помощью манипулятора setw.


26.Присваивание значений одной структурной переменной другой структурной переменной в С++…
Ответ: возможно только если эти переменные имеют один и тот же тип.


27. Сигнатурой функции называется…
Ответ: информация о количествах и типах аргументов функции.


28. Константный указатель в С++…
Ответ: не позволяет изменять значения переменной, на которую он указывает, это не то же, что и указатель на константу.


29. Функция fopen() в С++…
Ответ: связывает открытый файл с логическим файлом.


30. Типы формальных и фактических параметров функций в С++…
Ответ: могут не совпадать, но обязательно должны быть совместимы.


31. Среди представленных тезисов выберите те, которые не характеризуют языки программирования высокого уровня…
Ответ: Являются аппаратно-зависимыми, приближены к машинному языку.


32.Функции malloc() и calloc() в С++…
Ответ: различны по назначению, требуют не одинакового количества аргументов.


33. К способам описания алгоритмов относятся…
Ответ: блок-схемы, структурограммы, словесный.


34. Дружественные функции…
Ответ: имеют доступ к защищённым элементам класса, не являются элементами класса.


35. Компилятор в С++…
Ответ: выполняет проверку на соответствие количества и типов параметров при каждом обращении к функции, не игнорирует имена параметров функции.


36. Использование оператора goto…
Ответ: может использоваться не только при наличии метки, является допустимым, всегда противоречит парадигме структурного программирования.
